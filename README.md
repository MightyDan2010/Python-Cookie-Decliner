# Python-Cookie-Decliner

This script takes a list of websites (Information.json) and goes through every website trying to decline all the cookies.

**What it tracks:**
* Total audited sites
* Total number of cookies
* Sites where the code couldn't find the "Decline" button
* Total first-party cookies
* How many third-party cookies were used in websites where the code couldn't find the "Decline" button
* How many third-party cookies were used in websites where the code could find the "Decline" button
* A list of all third-party cookies that were used in websites where the code could find the "Decline" button
* How many cookies were active before and after the declining
